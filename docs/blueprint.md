# **App Name**: Temmy Realty Hub

## Core Features:

- Property Listings (Buy & Rent): Browse a comprehensive list of properties available for buying and renting, with distinct sections for each category.
- Advanced Property Search & Filtering: Utilize search functionality with filters for property type, price range, location, and other key amenities to find desired listings.
- Detailed Property Pages: View extensive details for each property, including high-quality images, descriptions, features, and contact information.
- Admin Authentication: Secure admin login and registration system to manage user roles and access to property management features.
- Admin Property Management (CRUD): Administrators can add, edit, update, and delete property listings (both for sale and rent) through a dedicated dashboard.
- AI-powered Listing Assistant: A tool to assist administrators in generating engaging and comprehensive property descriptions, ensuring consistency and attractiveness for new listings.
- Direct Contact System: Users can directly contact Temmy American Realty via email or phone/WhatsApp through prominently displayed contact information.

## Style Guidelines:

- Primary action color: A confident deep blue (#2273C3), evoking professionalism and trust in a competitive market.
- Background color: A very light, crisp blue-grey (#F0F2F5), providing a clean and expansive backdrop that feels fresh.
- Accent color: A vibrant cyan (#1AE6E6) to highlight calls to action and important information, ensuring strong contrast and clear visibility.
- Headlines: 'Space Grotesk' (sans-serif), for a modern, technically precise, and strong visual appeal suitable for main titles.
- Body Text: 'Inter' (sans-serif), providing excellent legibility and a neutral, objective feel for detailed property descriptions and general content.
- Utilize clean, line-art or minimalist vector icons that are easy to understand and maintain a modern, professional aesthetic across the site.
- Employ a responsive, grid-based layout for property listings, ensuring an organized and visually pleasing presentation on all devices, with ample whitespace for clarity.
- Implement subtle, fluid animations for page transitions and interactive elements, enhancing user experience without distracting from the content's professionalism.
